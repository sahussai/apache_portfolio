    • Portfolio

      • This portfolio was made for two purposes. Firstly, I wanted to have my own website up and running to experience all the different aspects of running a website. Because of that, I chose not to host this website using a professional hosting domain. Rather, I chose to host this website via Amazon Web Services (AWS), using their EC2 service. I got to play around with the inner workings of how the server is set up, and I got to decide which web server I was using. Currently, I am running this website on a Apache web server, but I plan to switch it to a Node.js server in the near future.

      I also wanted a platform where I could periodically collect my work in a more presentable fashion than just Github. As I complete more and more projects, I plan to display my results here for others and for myself.
